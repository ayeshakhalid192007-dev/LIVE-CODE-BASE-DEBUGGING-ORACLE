# MCP Tool Documentation

MCP tool documentation will be added in Phase 4 (Fix Generation and MCP Tool Contracts).

## Planned Tools

- `index_repo` — Full repository indexing
- `index_incremental` — Incremental indexing of changed files
- `debug_error` — Error debugging and fix generation
- `search_codebase` — Arbitrary code search
- `get_recent_diffs` — Recent commit diffs
- `get_index_status` — Index status query
